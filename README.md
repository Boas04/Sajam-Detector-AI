# Sajam-Detector-AI
Sajam Detector AI adalah sistem deteksi senjata tajam (Golok, Celurit, Parang) berbasis YOLOv11. Proyek ini mengotomatisasi pengawasan CCTV di ruang publik untuk mencegah kriminalitas seperti tawuran dan pembegalan. Menggunakan 450 data hasil augmentasi untuk akurasi dan presisi deteksi real-time yang optimal.
